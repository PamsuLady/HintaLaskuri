# HintaLaskuri

joka pyytää tuntipalkan, tehdyn tuntimäärän, materiaalikulut, toimitus-ja pakkauskulut sekä kateprosentin. 
Laskee:
- työkustannukset joihin kuuluu Yel ja muut lisäkulut (1.8%)
- materiaalikustannukset
- Omakustannushinnan
- Katteen määrän
- Verottoman hinnan
- Veron määrän (alv 24%)
- Kokonaishinnan
