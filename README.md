# HintaLaskuri

joka pyytää tuntipalkan, tehdyn tuntimäärän, materiaalikulut, toimitus-ja pakkauskulut sekä kateprosentin. 
Laskee:
- työkustannukset joihin kuuluu Yel ja muut lisäkulut.
- materiaalikustannukset
- Omakustannushinnan
- Katteen määrän
- Verottoman hinnan
- Veron määrän
- Kokonaishinnan
